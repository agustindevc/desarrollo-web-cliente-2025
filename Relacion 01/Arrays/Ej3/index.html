Guia 1 JavaScript - Ejercicio 3 - Arrays

Crea un script que pida al usuario un numero entero positivo N mayor a 2. Hay que controlar que el numero introducido sea correcto. Si no es así se volverá a pedir. A continuación, debe crearse una matriz NxN rellena con los resultados de la tabla de multiplicar de N (empezando en 1). Finalmente, muestra por consola la matriz “en forma de matriz”
