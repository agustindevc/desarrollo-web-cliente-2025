Aquí tienes una lista de las principales funciones y eventos que se pueden usar en JavaScript para interactuar con elementos HTML. Estos eventos permiten capturar diversas interacciones del usuario o cambios en la página. --- ### **Eventos de formulario**: 1. **`onfocus`**: Se activa cuando un elemento (como un campo de entrada) recibe el foco. 2. **`onblur`**: Se activa cuando un elemento pierde el foco. 3. **`onchange`**: Se activa cuando el valor de un elemento cambia y pierde el foco. 4. **`onsubmit`**: Se activa cuando se envía un formulario. 5. **`onreset`**: Se activa cuando se reinicia un formulario. 6. **`oninput`**: Se activa cuando el usuario escribe o cambia el valor en un campo de entrada. 7. **`onselect`**: Se activa cuando el usuario selecciona texto en un campo de entrada. --- ### **Eventos del mouse**: 1. **`onclick`**: Se activa cuando se hace clic en un elemento. 2. **`ondblclick`**: Se activa cuando se hace doble clic en un elemento. 3. **`onmousedown`**: Se activa cuando el botón del mouse se presiona sobre un elemento. 4. **`onmouseup`**: Se activa cuando se suelta el botón del mouse. 5. **`onmouseover`**: Se activa cuando el puntero del mouse pasa por encima de un elemento. 6. **`onmouseout`**: Se activa cuando el puntero del mouse sale de un elemento. 7. **`onmousemove`**: Se activa cuando el puntero del mouse se mueve sobre un elemento. 8. **`onmouseenter`**: Similar a `onmouseover`, pero no se activa en los elementos secundarios. 9. **`onmouseleave`**: Similar a `onmouseout`, pero no se activa en los elementos secundarios. --- ### **Eventos del teclado**: 1. **`onkeydown`**: Se activa cuando el usuario presiona una tecla. 2. **`onkeypress`**: (Obsoleto) Similar a `onkeydown`, pero no detecta teclas especiales como Shift o Alt. 3. **`onkeyup`**: Se activa cuando el usuario suelta una tecla. --- ### **Eventos de carga y recursos**: 1. **`onload`**: Se activa cuando una página o un recurso (como una imagen) ha terminado de cargarse. 2. **`onunload`**: Se activa cuando el usuario abandona la página. 3. **`onerror`**: Se activa cuando ocurre un error al cargar un recurso. 4. **`onresize`**: Se activa cuando se cambia el tamaño de la ventana del navegador. 5. **`onscroll`**: Se activa cuando el usuario desplaza la página o un elemento con barra de desplazamiento. --- ### **Eventos de arrastrar y soltar**: 1. **`ondrag`**: Se activa cuando un elemento es arrastrado. 2. **`ondragstart`**: Se activa al comenzar a arrastrar un elemento. 3. **`ondragend`**: Se activa cuando se completa la acción de arrastrar. 4. **`ondragover`**: Se activa cuando un elemento arrastrado está sobre otro. 5. **`ondrop`**: Se activa cuando un elemento arrastrado se suelta sobre otro. 6. **`ondragenter`**: Se activa cuando un elemento arrastrado entra en otro. 7. **`ondragleave`**: Se activa cuando un elemento arrastrado sale de otro. --- ### **Eventos de medios (audio y video)**: 1. **`onplay`**: Se activa cuando comienza la reproducción. 2. **`onpause`**: Se activa cuando la reproducción está en pausa. 3. **`onended`**: Se activa cuando la reproducción termina. 4. **`onvolumechange`**: Se activa cuando cambia el volumen. 5. **`ontimeupdate`**: Se activa durante la reproducción para actualizar el tiempo actual. 6. **`onseeked`**: Se activa cuando el usuario ha buscado un punto diferente en el video o audio. --- ### **Eventos del portapapeles**: 1. **`oncopy`**: Se activa cuando se copia contenido. 2. **`oncut`**: Se activa cuando se corta contenido. 3. **`onpaste`**: Se activa cuando se pega contenido. --- ### **Otros eventos útiles**: 1. **`oncontextmenu`**: Se activa cuando el usuario abre el menú contextual (clic derecho). 2. **`onfocusin`**: Se activa cuando un elemento o su descendiente gana el foco. 3. **`onfocusout`**: Se activa cuando un elemento o su descendiente pierde el foco. 4. **`onwheel`**: Se activa cuando el usuario usa la rueda del mouse. 5. **`ontouchstart`**: Se activa cuando se toca una pantalla táctil. 6. **`ontouchmove`**: Se activa cuando un dedo se mueve sobre la pantalla. 7. **`ontouchend`**: Se activa cuando un toque termina. --- ¿Quieres ejemplos de cómo usar algunos de estos eventos en JavaScript?
